# HodlHunt Manager

## Установка (один раз)

```bash
npm install
```

## Запуск для разработки

```bash
npm start                  # с терминалом
start-silent.vbs           # без терминала
```

## Сборка .exe для раздачи игрокам

```bash
npm run build
```

## Лицензионные ключи

Демо версия абсолютно бесплатна
За полной версией телеграмм @cmdRua 7 дней - 30 $
